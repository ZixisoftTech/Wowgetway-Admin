# Project Overview - Wow Gateway

Wow Gateway is an enterprise-grade premium travel, hospitality, tourism, transportation, homestay, hotel, sightseeing, booking, and logistics ecosystem.

## Technology Stack
- **Frontend:** React.js, Vite, Tailwind CSS v4, Framer Motion, Redux Toolkit, React Query
- **Backend:** Node.js, Express.js, JWT, BCrypt, Helmet, CORS, Sanitization, Nodemailer
- **Database:** MongoDB Atlas (Mongoose ODM)

## Folder Structure
- `frontend/`: Client applications and dashboards
- `backend/`: REST API server, security middlewares, and database schemas
- `docs/`: Self-documenting functional requirement documents, database structures, and Release notes
